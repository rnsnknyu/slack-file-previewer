# Slack File Previewer

ブラウザ版Slackで `.md` / `.html` ファイルをダウンロードせずにプレビューできるChrome拡張機能。

## 機能

- `.md` `.html` ファイルの横に **Preview** ボタンを自動表示
- モーダルでレンダリング済みのプレビューを表示
- Markdown: テーブル・コードブロック・引用などに対応
- HTML: サンドボックス化されたiframeで安全に表示

## インストール

1. このリポジトリをクローン
2. Chromeで `chrome://extensions` を開く
3. 「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」→ クローンしたフォルダを選択
5. Slackのタブをリロード
