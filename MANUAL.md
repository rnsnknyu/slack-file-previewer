# Slack File Previewer 利用マニュアル

**想定読者**: PortX社員（非エンジニア含む全員）
**最終更新**: 2026年4月2日

---

## このツールでできること

Slackに投稿された `.md`（Markdown）ファイルや `.html` ファイルを、**ダウンロードせずにその場でプレビュー**できるChrome拡張機能です。

ファイルカードの横に表示される **「Preview」ボタン** をクリックするだけで、整形された内容をモーダル画面で確認できます。

---

## インストール手順（所要時間: 約3分）

### ステップ1: ファイルをダウンロードする

1. 以下のURLにアクセスする
   - https://github.com/rnsnknyu/slack-file-previewer
2. 緑色の **「Code」ボタン** をクリックする
3. **「Download ZIP」** をクリックする
4. ダウンロードされたZIPファイルを展開（解凍）する
   - Macの場合: ダブルクリックで展開される
   - 展開先はどこでもOK（デスクトップ等）

> **補足**: 展開すると `slack-file-previewer-main` というフォルダができます。このフォルダを使います。

### ステップ2: Chromeに読み込む

1. Chromeのアドレスバーに `chrome://extensions` と入力してEnter
2. 右上の **「デベロッパーモード」をON** にする（トグルスイッチ）
3. 左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックする
4. ステップ1で展開した **`slack-file-previewer-main` フォルダ** を選択する
5. 拡張機能一覧にPortXロゴの **「Slack File Previewer」** が表示されれば完了

### ステップ3: 動作確認

1. ブラウザでSlack（app.slack.com）を開く
2. `.md` または `.html` ファイルが投稿されているチャンネルに移動する
3. ファイルカードの横に **紫色の「Preview」ボタン** が表示されていれば成功

```
インストールの流れ:

  GitHub        Chrome設定画面         Slack
  ──────        ──────────────         ─────
  ZIPを       → フォルダを読み込み  → Previewボタンが
  ダウンロード    (デベロッパーモード)    表示される
```

## 使い方

### 基本操作

| 操作 | やり方 |
|------|--------|
| プレビューを開く | ファイル横の **「Preview」ボタン** をクリック |
| プレビューを閉じる | **Escキー** / 右上の **×ボタン** / モーダル外をクリック |

### 対応ファイル形式

| ファイル形式 | プレビュー内容 |
|------------|--------------|
| `.md`（Markdown） | 見出し・箇条書き・表・コードブロック等を整形して表示 |
| `.html`（HTML） | Webページとしてレンダリングして表示 |

### 初回利用時の注意

Previewボタンを押す前に、**Slackを少し操作**してください（チャンネルの切り替え等）。これにより拡張機能がSlackの認証情報を取得します。

初回操作なしでPreviewを押すと「APIトークンが未取得」というエラーが出ることがあります。その場合はチャンネルを切り替えてから再度お試しください。

## 更新方法（新しいバージョンが出たとき）

1. GitHubから最新のZIPをダウンロードする（インストール手順のステップ1と同じ）
2. 古いフォルダを新しいフォルダに差し替える
3. `chrome://extensions` を開く
4. Slack File Previewerの **リロードボタン（🔄アイコン）** をクリックする
5. **Slackのタブをリロード** する（Ctrl+R / Cmd+R）

## トラブルシューティング

| 症状 | 原因と対処 |
|------|-----------|
| Previewボタンが表示されない | Slackタブをリロードしてください。それでも出ない場合は `chrome://extensions` で拡張機能が有効（トグルON）か確認してください |
| 「APIトークンが未取得」と出る | Slackでチャンネルを2〜3回切り替えてから再度Previewを押してください |
| 「ファイルのダウンロードに失敗」と出る | Slackタブをリロードしてから再試行してください |
| 「拡張機能が更新されました」と出る | 拡張機能をリロードした後、**Slackタブもリロード**してください |
| プレビューの読み込みが遅い | ファイルサイズが大きい場合は時間がかかることがあります。15秒でタイムアウトします |

上記で解決しない場合は、石田までご連絡ください。
